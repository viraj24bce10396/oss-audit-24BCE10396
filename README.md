## Project Description

This repository contains the shell scripts developed as part of the Open Source Software audit project. The scripts demonstrate practical Linux and shell scripting concepts such as system information retrieval, package inspection, disk auditing, log analysis, and user interaction.

---

## Shell Scripts Overview

### Script 1 — System Identity Report

Displays system information such as kernel version, user, uptime, and OS details.

### Script 2 — FOSS Package Inspector

Checks whether Python is installed and displays version, license, and description.

### Script 3 — Disk and Permission Auditor

Analyzes important system directories and shows their permissions and disk usage.

### Script 4 — Log File Analyzer

Reads a log file and counts occurrences of a keyword like "error".

### Script 5 — Open Source Manifesto Generator

Generates a personalized open-source philosophy statement based on user input.

## How to Run the Scripts

1. Open terminal in Linux

2. clone repository

3. Navigate to project folder:

   ```bash
   cd oss-audit-24BCE10396
   ```

4. Make scripts executable:

   ```bash
   chmod +x *.sh
   ```

5. Run scripts:

   ```bash
   ./script1.sh
   ./script2.sh
   ./script3.sh
   ./script4.sh /var/log/syslog error
   ./script5.sh
   ```
